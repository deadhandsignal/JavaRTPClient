Saving an RTP Stream

 

The user of the 'Cork Campus Radio RTP Admin' can save an RTP stream from either the 'Cork Campus Radio' RTP stream or a user specified RTP stream. The stream can be saved in either an uncompressed ('WAV') or compressed ('MP3') audio formats. To access this utility the user must choose the 'Save Stream' item in the 'Tools' menu of the 'Cork Campus Radio RTP Admin' main window. They are then presented with the 'Save RTP Stream' window shown in figure 7.1 below:

Figure 7.1 - 'Save RTP Stream..' window.

The user must firstly choose from what source they wish to save the stream from. Should they choose the CCR check box the parmeters for the CCR stream will be used for the Save Stream function. Should they choose the 'Other Rtp' check box they will be presented with text areas to enter parameters for an RTP stream.

The user then chooses the destination and file name for the new audio file by choosing the 'Select' button in the second panel of the 'Save RTP Stream..'. The user is then presented with a 'File Chooser' window from which they nominate where they want to save the audio file and what they want to call it (figure 7.2).

Figure 7.2 - 'Save RTP Stream' destination 'File Chooser'.

Once a destination file has been selected the user then selects the duration they wish to save the stream in panel three of the 'Save RTP Stream..' window in fig. 7.1. When the save stream parameters have been selected correctly the user should then select the 'Save' button in panel 4 of the 'Save RTP Stream..' window in fig. 7.1 above. A progress bar will then show stream saving, giving the user a percentage figure of how much has completed in the particular saving operation, see fig 7.3 below.

 

Figure 7.3 - 'Save RTP Stream..' window showing 'Saving' progress bar.
