Editing the CCR Schedule

 

The 'Edit Schedule' utility allows members of the radio station to change the programme schedule that will be viewed by users of the 'Cork Campus Radio RTP Client' version of my application. To access this utility the user must choose the 'Edit/Display' schedule item in the 'Tools' menu of the 'Cork Campus Radio RTP Admin' main window. They are then presented with the simple window shown in figure 5.1 below:

 

Figure 5.1 - Database utility 'chooser' window.

By choosing the 'Edit Schedule' button, the user will be presented with the 'Edit Schedule..' window shown in figure 5.2 below:

Figure 5.2 - 'Edit Schedule..' window.

To edit the schedule the user must first choose the time and day of the programme they wish to edit. This is done with the drop down menu items in the first panel. Once this has been done the user must select the 'OK' button, this action will display any programme and it's information and presenters in the text areas in panel 2. Should there be no programme for the selected time or day, the text areas will be empty.

The user must then place the values that they require in the text areas provided in panel two replacing any values that may or may not already exist for that time and day. To update the schedule the user then must choose the 'Update Schedule' button at the bottom of the window.

To view the changes made to the schedule, to insure they are correct the user can display the schedule for the day of the week for which they have made the alteration with the 'View Change' button.
